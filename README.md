# ServiceHub — Find Trusted Local Professionals

A platform that connects users with local service providers like plumbers, electricians, cleaners, and other home service professionals. Built with Next.js and deployed on Vercel.

## Live Demo

[public-service-chi.vercel.app](https://public-service-chi.vercel.app)

## Features

- Browse local service providers by category
- User authentication — sign up and log in
- Service providers can list and manage their services
- Contact and connect with providers directly
- Clean, responsive UI

## Tech Stack

- Next.js
- JavaScript
- Tailwind CSS

## Getting Started

```bash
git clone https://github.com/Mehdi-Chandia/PublicService.git
cd PublicService
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
